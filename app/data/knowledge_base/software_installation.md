# Software Installation Request Policy

## Overview
All software installations on company devices must be requested and approved through the IT Portal. Unauthorized software installations are a violation of the Information Security Policy.

## Pre-Approved Software
The following software is pre-approved and can be installed via the Company Software Center without a ticket:
- Microsoft Office 365 Suite
- Google Chrome / Mozilla Firefox
- Slack Desktop
- Zoom Client
- Visual Studio Code
- Git
- 7-Zip
- Adobe Acrobat Reader

## Requesting New Software
1. **Submit a request**: Go to https://itportal.company.com/software and fill out the Software Request Form.
2. **Required information**: Software name, version, business justification, license type (free/paid), and vendor URL.
3. **Security review**: The IT Security team reviews all requests for vulnerabilities and compliance within 3 business days.
4. **License procurement**: If approved, IT Procurement acquires licenses (if paid). Budget must be pre-approved by department head for costs over $500.
5. **Installation**: IT installs the software remotely or provides self-install instructions within 2 business days of approval.

## Denied Requests
Common reasons for denial:
- Software has known security vulnerabilities.
- A pre-approved alternative exists with equivalent functionality.
- Software requires admin privileges that conflict with the least-privilege policy.
- License cost exceeds department budget.

## Open Source Software
- Open-source software is permitted if it has an approved license (MIT, Apache 2.0, BSD).
- GPL-licensed software requires additional legal review.
- Submit open-source requests through the standard process above.
